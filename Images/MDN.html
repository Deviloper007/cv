AV1 & Video

AV1 is a royalty-free open codec for transmitting high-quality video over the internet, created by Mozilla and the Alliance for Open Media.

Browser Extensions

Extensions allow developers to add features to browsers and empower users to customize their browsing experience. Extensions for Firefox are built using the WebExtensions API, a Web-based framework that offers a powerful way to build extensions that can easily be ported to Chrome, Opera, and Edge.

CSS

CSS is a declarative stylesheet language used to describe the layout and presentation of HTML documents.

Developer Tools

Mozilla's Developer Tools help developers examine, edit, and debug HTML, CSS, and JavaScript.

Firefox for Android

Get the development tools you need to build mobile browsers and web applications. Mozilla uses GeckoView to power Firefox Reality, Firefox Focus, and other Android apps. GeckoView serves a similar purpose to Android's built-in WebView, but it has its own APIs and is not a drop in replacement.

JavaScript

Javascript is a lightweight, dynamic scripting language for the web that supports interactivity and is an essential part of web applications.

Mixed Reality / WebXR

WebXR is a new web standard for virtual and augmented reality on the open web. You can add VR and AR content to your existing pages, or create brand new immersive experiences, and know that it will just work on the web. No native code required.

Read more to get started.

Rust

Rust is a multi-paradigm systems programming language syntactically similar to C++. It is focused on safe concurrency and high performance.

Voice

A portfolio of open source tools and resources supporting the development of voice interactivity on the web and beyond.

WebAssembly

WebAssembly (WASM) is a low-level assembly-like language with a compact binary format that runs with near-native performance.

WebThings

WebThings is an open IoT platform from Mozilla for monitoring and controlling devices over the web. It includes the open source WebThings Gateway and Framework.
